# SPACEPAN MODULE - README

    ##FUNCTIONALITY:
    - Remaps MButton to Space + LButton.
    - Specifically designed for Adobe Suite panning.

    ## REQUIREMENTS:
    - Must have a global 'Toggles' object defined in Main.ahk.
    - Only works when 'SpacePan' key is set to 1.

    COMPATIBLE APPS:
    1. Photoshop
    2. InDesign
    3. Acrobat / Reader
    4. Muse
